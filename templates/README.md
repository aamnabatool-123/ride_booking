# Ride Booking Website 🚗

Welcome to the **Ride Booking Website**, a full-stack web application designed to simulate a real-time ride-sharing and booking platform.

## 🚀 Features
- **User Authentication:** Secure signup and login functionality for riders.
- **Ride Booking:** Seamless interface to request, book, and track rides.
- **Dynamic Dashboard:** A responsive dashboard rendering real-time ride details and booking history.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Flask (Python)
- **Database:** MongoDB

## 📋 Prerequisites
To run this project locally, make sure you have Python and MongoDB installed and running on your system.
